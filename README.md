# luckycfg (ru/en)
config for DDrace Network
# **(ru) как использовать**
> [!TIP]
> Вы можете устанавливать конфиг не выходя из игры.

если через стим:
1) открываете библиотеку
2) находите игру
3) кнопка настроек (управление)
4) пункт управление
5) кнопка посмотреть локальные файлы
6) после нажатия кнопки, вы увидите папку ddnet (это корневая папка игры)
7) скачиваете конфиг
8) в папке конфиг есть папка luckymod и файл start, их нужно перекинуть в корневую папку игры(папка ddnet которую мы открывали ранее)
9) после успешного переноса можно заходить в игру (если выходили)
10) заходим на любую карту
11) нажимаем f1 (должна открыться консоль)
12) пишем туда:
```
exec start
```
13) после чего в чат вам должно высветится сообщение об успешной установке
14) по нажатии f11 можно будет увидеть все бинды
> [!NOTE]
> Вы можете настраивать бинды под себя в файле start.txt
